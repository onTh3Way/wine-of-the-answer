-----------

#### 功能

> 获取年龄统计情况

#### URL

> GET /api/v1/stat/ages

#### 返回字段
| 返回字段 | 字段类型 | 说明     |
| :------- | :------- | :------- |
| ages     | int      | 年龄     |
| num      | int      | 所占人数 |

#### 响应
##### 获取成功: 200
JSON
```JSON
[
  {
    "ages": 20,
    "num": 2000
  },
  {
    "ages": 20,
    "num": 2000
  }
]
```
