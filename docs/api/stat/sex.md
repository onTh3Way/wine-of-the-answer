-----------

#### 功能

> 获取性别统计情况

#### URL

> GET /api/v1/stat/sex

#### 返回字段
| 返回字段 | 字段类型 | 说明                      |
| :------- | :------- | :------------------------ |
| sex      | int      | 性别: 1为男,2为女,0为未知 |
| num      | int      | 所占人数                  |

#### 响应
##### 获取成功: 200
JSON
```JSON
[
  {
    "sex": 2,
    "num": 2000
  },
  {
    "sex": 1,
    "num": 2000
  }
]
```
