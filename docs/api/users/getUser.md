-----------

#### 功能

> 获取用户信息

#### URL

> GET /api/v1/users/{id}

#### 返回字段
|返回字段|字段类型|说明 |
|:----- |:------|:----------------------------- |
|nickname | string | 用户名 |
|avatar | string | 头像地址,用户无头像时此项为空 |
|freezeTime | string | 冻结截止时间 |

#### 响应：
##### 成功：200
响应格式：JSON
```
{
	"nickname": "phantom",
	"avatar": "blbla.png",
	"freezeTime": 1111111
}
```
##### 未登录: 401
##### 权限不足(不是管理员无法获取别人信息): 403
##### 用户不存在: 404
