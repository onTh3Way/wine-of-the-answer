# 管理员专属
#### 功能

> 获取用户信息

#### URL

> GET /api/users?offset=0&limit=5

#### 请求参数

|参数|必选|类型|默认值|说明|
|:----- |:-------|:------|:-----|----- |
|offset|false|int|0| 从第几个开始|
|limit|false|int|0| 获取几个|

#### 返回字段
|返回字段|字段类型|说明 |
|:----- |:------|:----------------------------- |
|users | Array\<user> | 用户列表 |
|total | int | 用户数量 |

#### 响应：
##### 成功：200 <br>
响应格式：JSON
```
	{
		users: [{// 数据同getUser的数据}],
		total: 400
	}
```
##### 权限不足: 401
